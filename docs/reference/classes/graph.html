Examples

>>> G.add_vertex('A')

>>> v3 = G.add_vertex() # automaticaly label the vertex v3 as '3'
>>> G.num_vertices()

Graph.add_edge(*args, **kwargs)¶

Overloaded function.

  1. add_edge(self: pybstream.Graph, source: int, target: int) -> pybstream.edge_t
  2. add_edge(self: pybstream.Graph, source: str, target: str) -> pybstream.edge_t

Add an edge between s and t.

Parameters:
  • source (int, str) – source vertex, could be a vertex id or a vertex label
  • target (int, str) – target vertex, could be a vertex id or a vertex label
Returns:

edge_t – edge

Examples

>>> G = pb.Graph()  # or DiGraph, etc
>>> G.add_vertex('A')
>>> G.add_vertex('B')
>>> G.add_adge('A', 'B')
>>> G.num_edges()
1
>>> G = pb.Graph()  # or DiGraph, etc
>>> s = G.add_vertex()
>>> t = G.add_vertex()
>>> G.add_adge(s, t)
>>> G.num_edges()
1
