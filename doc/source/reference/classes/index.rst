***********
Graph types
***********

Pybstream provides data structures and methods for storing graphs.

The choice of graph class depends on the structure of the
graph you want to represent.

Which graph class should I use?
===============================

===================  ========================
Graph Type           Pybstream Class
===================  ========================
Undirected Simple    Graph
Directed Simple      DiGraph
Bipartite Simple     Bipartite
===================  ========================
